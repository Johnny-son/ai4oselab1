# Work（你的教学实验环境）

这里放你**最终要交付**的“适合自己的教学实验环境”。

## 你需要逐步具备的内容

- 实验指导文档（Markdown，可配 mermaid 图）
- 实验代码（Rust，目标 RISC-V 64，可在 QEMU 上运行）
- 测试用例（单元测试/系统测试）
- 组件化：把可复用模块拆成 crate（必要时发布到 crates.io）

## 推荐组织方式

- `os/`：内核/实验主体（可做成 workspace）
- `crates/`：拆出来的可复用模块（例如 console、vm、syscall 等）

> 说明：你也可以先把参考仓库跑通，再逐步把改进后的实现迁移到这里。
